# AI教育推進機構 学生部 公式サイト

このファイルは制作の常設ルールである。作業前に必ず読み、ここに反する出力をしない。
詳細仕様（head テンプレ・OGP・JSON-LD・SEO手順・デプロイ）は `SITE_SPEC.md` を参照する。

## このプロジェクトは何か
- AI教育推進機構 学生部の公式サイト。本部(`ai-ready-education.org`)の**下位ブランド**。
- 目的は**対外PR・実績紹介**。第一の読み手は提携先・一般、第二に参加希望の学生。
- 狙うトーンは「信頼できる先進性」。**本部ロゴに合わせた白基調 × 濃紺 × ブルーのオフィシャルな佇まい**を基本とし、温かさは写真・文章で出す（冷たいだけのテック感は避ける）。本部・学生部・個人サイトで一貫した公式トーンを共有する。

## スタック（厳守）
- **純粋な HTML / CSS / JS のみ。ビルド工程なし。フレームワーク不使用。**
- GitHub Pages で公開。1ファイル1ページ。共通の `css/style.css` と `js/main.js` を全ページで読む。
- 色・余白・角丸はすべて `:root` の CSS変数経由で使う。直値の手打ち禁止。

## 絶対にやらないこと（最重要）
「AIが量産したテンプレ」に見せないための禁止事項。違反は差し戻し対象。
- ❌ ピル型（`border-radius:100px`）のバッジ・ラベル。角丸は **4〜6px** に限定。
- ❌ グラデーション背景・放射状ブロブ・斜め clip-path。
- ❌ 常時表示のグラデ上線・光るドット・強い影・hover時の大きな拡大。
- ❌ 大文字・超ワイド letter-spacing ラベルの多用。
- ❌ **アクセント色を増やすこと。アクセントはブルー一色のみ（本部ロゴ準拠 `#1f6ab0`）。**
- ✅ 代わりに: フラットな配色・控えめな角丸・タイポの強弱・ブルーの一点投入・ホバー時のみの控えめなアクセント線。

## ページ構成
```
index.html       トップ（ヒーロー / 実績ハイライト / News / CTA）
about.html       私たちについて（理念・価値観3つ・設立背景・目指す姿）
activities.html  活動・実績【中心】（note / YouTube学習ガイド / 選書 / イベント）
members.html     体制・メンバー（運営学生 + 社会人/専門家の支援）
contact.html     お問い合わせ・参加（提携窓口 / メンバー募集）
```
- ナビ項目は全ページで統一: `活動・実績 / 私たちについて / 体制 / お問い合わせ`。
  ページ追加時は全ファイルのナビを更新する。
- お問い合わせのみブルーのボタン扱いにして主CTAを際立たせる。
- 学習リソース（選書・学習導線）は v1 では `activities.html` 内のセクション。

## インタラクションの統一言語
「ラインがホバーで出現」をサイト全体の統一モチーフにする（色はブルー）。
- カード: ホバーで上部ブルー線 + `translateY(-3px)` + 控えめな影。
- 項目: ホバーで左にブルー縦線（`scaleY`で出現、レイアウトずれなし）。

## 文章・コンテンツの規約
- `lang="ja"`、セマンティック HTML、アクセシビリティに配慮。
- レポート的・論理的な文章は「だ・である」。UI文言は自然な常体。
- 事実は正確に。数値・日付・期間は**推測で書かず、ユーザーに確認**して確定する。
- 日付は絶対表記（「昨年」ではなく「2025.08」）。News・実績は新しい順。
- 煽り文は控えめに。英語ラベル + 日本語見出しの組み合わせを基調とする。
- 実績は誇張しない。活動は **「実施中／構想中」を明示**する（価値観=信頼の体現）。
- 「提携でできること」をトップと contact に最低3点、具体で置く（現状の空白を埋める）。
